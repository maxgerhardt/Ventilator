Detailed Description

Abstract base class for debug variables.

This class represents a variable that you can read/write using the debug serial port. These are also variables that can be recorded in traces.

We give each such variable a name which the debugger command line will use to access it. We can also link it with a C++ variable whose value it will read or write.

Each variable will register itself with the debug variable Registry, which will issue it a unique ID. However, name uniqueness is not automatically guaranteed, and you are responsible for ensuring it, lest the client side of the debugger fail to function properly.
